## What is an Undirected Graph?
An undirected graph is a graph in which each edge has no direction. If an edge connects vertices $u$ and $v$, then the connection can be traversed both ways, so $(u,v)$ and $(v,u)$ represent the same edge.

Undirected graphs are used when relationships are mutual, such as friendship networks, physical roads between intersections, or communication links that work in both directions.
